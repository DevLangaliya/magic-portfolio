## Overview

Developed a React Native mobile application with a Python backend to help university students manage schedules and receive personalized course recommendations.

## Key Features

- **Recommendation Engine**: Implemented cosine similarity algorithm to generate personalized course suggestions.
- **Data Management**: Leveraged Supabase and PostgreSQL to handle 10 different user data types with scalable endpoints.
- **Cross-Platform Development**: Created a mobile interface compatible with multiple platforms.

## Technologies Used

- **React Native**
- **Python**
- **Flask**
- **scikit-learn**
- **PostgreSQL**
- **Supabase**
